CAFfeZeit

CAFfe-Zeit ist ein Projekt, dass aus den vielen Trainings, Hackathons und Workshops rund um das Microsoft Cloud Adoption Framework CAF in den letzten zwölf Monaten entstanden ist. Das CAF ist ein so umfangreiches Werk und es gibt so viele Aspekte rundum das CAF. Viele Teilnehmer aus meinen Trainings haben die Idee einer “CAF Runde” mitgetragen. Daher habe ich die CAFfe-Zeit ins Leben gerufen. Eine Community von CAF Interessierten, die sich regelmäßig austauschen, Diskussionen führen oder auch einfach ein paar News konsumieren möchten. Dafür habe ich diese kleine Website aufgesetzt um die Ergebnisse dieser CAFfe-Zeit bereitzustellen.

Ich habe bereits eine kleine Umfrage unter den Teilnehmern meiner Trainings gemacht und ein erstes Backlog von Themen für die CAFfe-Zeit aufgebaut. “Stay tuned” erst Inhalte werden also kommen…

Gern nehme ich ins Backlog noch weitere Ideen, Themen oder Fragen auf! Geht dazu auf das öffentliche CAFfeZeit Repo Diskussion auf Github und nehmt an der Diskussion teil. Aktive Mitglieder der Community werden wir auch gern immer wieder einladen, Post zu schreiben, Videos aufzunehmen oder einfach auch nur als Quelle für eine Idee genannt zu werden.

Ich freuen uns auf Eure aktive Mitwirkung!

Viel Spaß
